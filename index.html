Who are we?

We are Lavender Labs: A group of high school developers honing our CS skills by making diverse projects. As childhood friends, it’s our mission to improve as a team and achieve our goal of developing innovative games. We met in 2018 while playing video games with mutual friends (and still do our Friday game nights!) Even early on, our individual passions for programming were obvious; but as a team, we’ve helped and inspired each other to improve far beyond our capabilities alone. Currently, the team is focused on building our foundation by working on projects and improving our skill sets. We hope you enjoy browsing through our projects from throughout the years!

Projects

Chess (For Beginners)

Using the most basic coding concepts to replicate a not so basic board game.

Conway's Game of Life

We recreated Conways Game of Life to familiarize ourselves with 2D arrays. Also helped us familarize ourselves with a new coding language, Swift. Programmed in Swift.

EZ Statistics Project

A statistical project to make operations like finding mean, median, standard deviation, and primes an easy task. We created a UI to allow users to add their own data sets and store them to files using for later use. Implented in Swift.

Turing Virtual Machine

We created a working virtual machine based off of the Turing machine architecture created by Alan Turing. This project helped us as an introduction to virtual machines. Developed using Swift.

Huffman Tree Encoding

Used binary trees to create a Huffman tree, a way of encoding messages efficiently. Helped us familiarize ourselves with binary trees and inheritance. Coded in Java.

Assistant App

We created an application to help with organization. The app sets reminders, tells the weather, create notes, get current news articles, and more using API's. Coded using Java.

Benchmarking Linear, Binary, and Hash maps

We implemented our own map data structures (linear, binary, and hash) to understand basic benchmarking and time efficiency. Also helped us understand the innerworkings of various data structures. Coded using Swift.

Pathfinder

We created a game similar to Packman in which users can create a maze and avoid the ghosts. The 3 ghost AI use depth first search, breadth first search, and randomized search that allow them to track the player. Coded using Java.

Simple Assembly Program

A complex virtual machine based off of the architecture of the PDP-11. Uses assembly language and includes a compiler, memory, debugger, and UI. Coded using Swift.

Newton Public Lunch Tracker

For the NNHSHacks 2021, we used javascript and puppeteer, to scrape nutritional info from our school's lunch provider. We also used Next.js to create a website to compile this information on.

Minesweeper Bot

Created a minesweeper Discord bot using node.js and discord.js, which users could interact with by sending messages over discord to play minesweeper!

MC Yelp

A discord bot used to moderate, manage, and automate a yelp service for businesses on a minecraft server we played on, in a server with over 100 people. Built using node.js and discord.js.
